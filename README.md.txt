# 🎬 Movie Analytics Project (Power BI + PostgreSQL)

A data analytics project analyzing over 1 million rows of movie data using PostgreSQL for data wrangling and Power BI for visual storytelling.

---

## 📊 Tools & Technologies

- **Power BI Desktop** – For interactive dashboard creation
- **PostgreSQL** – For advanced SQL queries
- **Excel** – Raw dataset (10 lakh rows)
- **DAX** – For dynamic calculations in Power BI

---

## 📁 Project Structure

| Folder | Description |
|--------|-------------|
| `SQL/` | Contains 10 advanced PostgreSQL queries used for data analysis |
| `PowerBI/` | `.pbix` file (Power BI dashboard) |
| `Exports/` | Screenshots or exported PDF of the dashboard (for demo) |
| `Dataset/` | Sample dataset (if public) |

---

## 🧠 Key Analytical Insights (SQL)

1. **Top 10 Movies by ROI**
2. **Genre-Wise Avg IMDb Rating & Revenue**
3. **Top Directors by Avg Revenue (min 5 movies)**
4. **Release Trends by Year & Genre**
5. **IMDb Rating vs Revenue Buckets**
6. **Frequent Lead Actors with Avg Revenue & Rating**
7. **Critically Acclaimed but Commercial Flops**
8. **Decade-wise Genre Popularity**
9. **Most Profitable Movie Each Year**
10. **Top Director-Actor Collaborations**

## 📤 How to Use

- Download the `.pbix` file from `PowerBI/` and open it in Power BI Desktop
- View or edit the SQL queries in `SQL/`
- Optional: Try it with your own movie dataset

---

## 🚀 What You’ll Learn

- Advanced SQL queries (Window functions, Aggregations, CASE logic)
- Building insightful dashboards in Power BI
- Applying DAX for interactive visuals
- Business thinking behind analytics (profitability, rating trends, talent performance)
